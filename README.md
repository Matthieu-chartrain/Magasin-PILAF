# Magasin-PILAF
Projet réalisé en fin de première année de DUT Informatique dans le cadre d'un travail de gestion de projet. Ce projet a été réalisé en collaboration avec Sylvain Frediani, Sylvain Masclet, Kevin Ayrault et Maxime Nguyen en HTML/CSS.

## Contexte
En fin d'année de première année de DUT Informatique, nous devions réaliser un travail dans la matière de gestion de projet en groupe. Le but de ce travail était de voir comment gérer un projet à plusieurs, les rôles à attribuer, l'estimation du temps de travail. Pour cela, nous avions un projet fictif à réaliser, un ERP (Enterprise Resource Planning) pour un magasin. Ce projet devait permettre de voir et de gérer l'information du magasin. Nous avions donc un client fictif, Mr PILAF, joué par notre professeur. Nous devions donc réaliser des pages précises de cette ERP, en HTML/CSS. Nous n'avions pas à créer une base de donnée, ni à rendre le site opérationnel, le but était simplement de voir comment gérer un projet.

## Installation
Pour essayer ce projet, Vous n'avez qu'à télécharger le code et à ouvrir "index.html" dans un navigateur. 

## Principe
Vous devriez arriver à une page de connexion. Pas d'inquiétude, le site n'est lié à aucune base de donnée, ce qui signifie qu'il n'y a rien d'important à mettre dedans. Remplissez juste les zones de textes comme vous le voulez et valider. Vous arrivez alors à l'accueil. De là, vous pouvez accéder à toutes les pages du site, en utilisant le menu. Vous pouvez créer des produits ou réceptionner une livraison, cela ne fera rien de plus qu'un message indiquant que l'opération a réussi. Comme il n'y a que du HTML/CSS, il n'y a que l'affichage des éléments qui est pris en compte.
